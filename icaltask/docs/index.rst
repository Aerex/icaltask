icaltask
====================================

``icaltask` is a commandline tool for synchronizing taskwarrior tasks with iCal server instance as VTODO tasks. The tool will generate the necessary UDA fields on your taskwarrior configuration. In addition, a on-launch and on-modify hooks will be created.
